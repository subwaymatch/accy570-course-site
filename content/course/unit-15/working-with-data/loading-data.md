Loading data is the process of importing data from somewhere into a software package for analysis. You have already been doing this for a few weeks in Tableau.

**Common data sources**

- Excel files (`.xlsx`)
- CSV files (`.csv`)
- Databases

**Common software for data analysis:**

- Excel
- Python / Pandas
- R
- Power BI
- Tableau
